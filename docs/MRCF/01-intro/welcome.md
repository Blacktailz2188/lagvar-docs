# 🚀 MRCF 템플릿 소개

:::tip 팁
이 템플릿은 최신 React 버전에서 가장 잘 작동합니다!
:::

이 문서는 제 Fab 템플릿의 첫 번째 문서입니다!

## 주요 기능
- **빠른 속도**: 최적화된 코드로 제작되었습니다.
- **쉬운 설정**: 누구나 5분 만에 시작할 수 있습니다.

:::danger 주의사항
`config.js` 파일을 삭제하면 작동하지 않으니 주의하세요!
:::
